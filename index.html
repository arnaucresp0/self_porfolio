PROJECTS

ROS2 navigation with a quadruped robot

This project integrates autonomous navigation tools using the ROS2 framework into the WAVEGO quadruped robot. A Raspberry Pi 4, LIDAR, and ROS2 were added, along with a 3D robot model, a 3D-printed LIDAR mount, and a ROS2 environment configured on a virtual remote machine. Effective communication between the robot and the virtual machine was established to enable navigation and control through custom nodes. By incorporating LIDAR-based perception, the project enhances the robot's navigation and control, enabling potential applications in tasks like inspection and surveillance in enclosed environments.

Educative arm robot with Lego

This project developed a robotic arm using Lego Education and a custom web interface to simulate chemical experiments, aiming to inspire children, especially girls, to explore technology, programming, and robotics. The robot offers extensive movement and mimics industrial robots, while the web interface provides dual functionality: managing access and usage for teachers or event coordinators, and offering intuitive controls, mode selection, and simplified programming for kids. This approach aims to make the experience engaging and encourage interest in technological fields.

Custom ESP32-Based PCB for EMG-Controlled Prosthetic Arm

This project involved designing a custom PCB featuring an ESP32 microcontroller to control brushless motors based on signals from EMG sensors. The system enables precise control of a prosthetic arm, translating muscle activity into smooth and responsive movements.

IoT plant monitoring

This project is designed to monitor the environmental conditions of a plant through a website and send alert messages via the WhatsApp API when critical conditions are detected. This project leverages an ESP8266-based web server to collect and display real-time sensor data, allowing plant owners to remotely track the well-being of their plants.
